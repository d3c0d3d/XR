# XR - Experimental Runtime

Runtime Experimental that downloads/compile and executes C# codes modified named (.xr) in a decentralized way from a single file or url

## Usage

### Example
```
run=https://pastebin.com/raw/CwicAkSV
```
or
```
run=file.xr
```
